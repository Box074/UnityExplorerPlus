# Unity Explorer Plus

[![Build](https://github.com/HKLab/UnityExplorerPlus/actions/workflows/build.yml/badge.svg)](https://github.com/HKLab/UnityExplorerPlus/actions/workflows/build.yml)

## Features

1. Allows finding Enemy with Mouse Inspector
2. Added Mods panel to help developers quickly find Mod instances
3. Show FsmName behind PlayMakerFSM component in GameObject view of UntiyExplorer Inspector panel
4. Easier to inspect Fsm in Inspector panel
5. Makes finding Prefabs easier
6. Allow dumping of tk2dSprite Animation from the UnityExplorer Inspector panel
7. Allow dumping of UnityEngine.Sprite as png from the UntiyExplorer Inspector panel

## Work with other mods

### [Satchel](https://github.com/PrashantMohta/Satchel)

1. Allows dumping PlayMakerFSM from the UntiyExplorer Inspector panel

## How to Build

1. Clone repo
2. Run `npm install` in the project root directory
3. Run `npm run csproj` to generate `UnityExplorerPlus.csproj`
4. Run `npm build` to build
